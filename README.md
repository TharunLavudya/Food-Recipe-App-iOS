# Food Recipe App 

The **Food Recipe App** is a mobile application developed using **Swift and SwiftUI** that helps users discover, explore, and cook a variety of recipes easily. The application provides detailed information for each recipe, including ingredients, preparation steps, and cooking instructions, making it easier for users to follow along while preparing dishes. It focuses on providing a simple and user-friendly cooking experience.

### Recipe Browsing
Users can browse through different recipes and explore various dishes available in the application.

The home screen can also recommend recipes based on the **user's selected interests**, making the experience more personalized.

### Recipe Details
Each recipe contains:
- Recipe name
- Dish image
- Ingredients list
- Step-by-step cooking instructions
- Preparation and cooking time
- The application also suggests **similar recipes within the recipe detail page** based on the meal type under section More Like This. This helps users discover other related dishes they may want to try.

Users can clearly view all the information required to prepare the dish.

### Search and Filter

Users can search for recipes by name and filter them based on multiple criteria such as **Cooking Time, Difficulty Level, Rating, Cuisine, and Meal Type**.

Meal types include:
- Breakfast
- Lunch
- Dinner
- Snacks

These options help users quickly discover recipes that match their preferences, available time, and desired cuisine.

### Profile View
The **Profile section** allows users to manage their personal information and preferences.

Users can:
- View their profile information
- Edit their **username**
- Update their **description**
- Select or update their **gender**
- Add their **food interests**
- Edit published recipes

Recipes created or added by the user are displayed in the **profile section**.

The application also uses the user's selected **interests to personalize the Home screen**, showing recipes that match the user's preferences.
